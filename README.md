# yolov8-oracle-detect

## **一个基于 C++ & Python 的 yolov8 识别部署实践**

---

**请确保有以下C++环境**(版本相关保持最新即可使用)
+ `OpenCV`
+ `CMake`
+ 支持 `C++17` 及以上的编译器

**请确保有以下Python环境**(版本相关保持最新即可)
+ `ultralytics`
+ `tqdm`
+ `fastapi`
+ `shutil`
+ `uvicorn`
+ `opencv-python`
+ `onnxruntime`
+ `numpy`
+ `pydantic`
---

以上`Python`模块使用`pip`或`conda`安装即可。你也可以到python文件夹内使用requirements.txt
